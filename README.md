## 关于我

我是蔚蓝（Leo），04 年出生，浙江农林大学计算机科学与技术专业本科毕业，现在在杭州做 AI 产品经理。

工作之外，我还是一个停不下来的独立开发者：写代码、做设计、研究 Agent，把脑海里的 idea 做成真的能跑起来的产品。最近也在认真做内容，希望把自己的真实实践、踩坑和思考毫无保留地分享出来，帮更多人跨过 Vibe Coding 的门槛。

我相信，AI 不该只制造焦虑，它更应该把人从重复劳动里解放出来。**你的 idea，才是定义生产力的唯一尺度。**

```text
idea → AI + 代码 → 产品 → 分享 → 真实反馈 → 下一次做得更好
```
